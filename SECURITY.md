# Security Policy

This policy outlines how we handle security for the **Nova Suite** project. 

## Supported Versions

Security fixes are prioritized and may trigger an immediate patch release outside of the normal release cycle.

## Reporting a Vulnerability

We appreciate responsible disclosure. If you discover a vulnerability, please help us keep Nova Suite secure.

### How to Report

The preferred and most secure way to report a vulnerability is via the **"Report a vulnerability"** button under the **Security** tab of this GitHub repository. 

This opens a private advisory between you and the maintainers. We aim to respond to all valid reports within **48 hours**.

---
*Note: For issues regarding 3rd-party dependencies, we use Dependabot and manual overrides to keep the core clean. If you find an unpatched dependency, feel free to open a regular Issue or PR.*
